# Class Average Calculator

## Author: Shai Scheinson

## Date: 02/19/2026


### Description
A beginner Python project that asks user for a number of students and their grades, then calculates the class average.

### How to Run
1. Make sure Python 3 is installed.
2. Download or clone the repository.
3. Run in terminal:
```bash
python3 class_average_calculator.py
```
